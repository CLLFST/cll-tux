# Cll-Tux
Cll tux running for freedom

![Alt text](/Screenshot.png?raw=true "Cll-Tux")

### Requiremnets:
  - Pygame: ```$ apt-get install python-pygame```

Clone the repo:
```$ git clone https://github.com/CLLFST/cll-tux.git ```


Then go to the game directory
 ```$ cd gamepath```

Execute main.py
```$ python main.py```
